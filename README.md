# WH40K Brasil


In the grim darkness of the far future, there is only war.

Projeto de site de fã sobre o universo de **Warhammer 40.000**, feito para estudar e praticar 
desenvolvimento web (HTML, CSS e JavaScript) enquanto organiza facções, personagens e recursos do cenário.

## ✨ Funcionalidades

- Página inicial com introdução ao universo de Warhammer 40K.
- Seção de facções com cards ilustrados e navegação por setas.
- Sidebar com lista de personagens e detalhes básicos.
- Mapa da galáxia em imagem destacada.
- Seção “Sobre o Projeto” explicando o objetivo do site.
- Layout responsivo em evolução para funcionar bem em desktop e mobile.
